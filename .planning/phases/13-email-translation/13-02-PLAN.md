New interface addition:
```typescript
interface SessionSummaryLabels {
  heading: string;        // "Session #{sessionNumber} Summary" — already interpolated by caller
  greeting: string;       // "Hi {recipientName}, here is the summary..." — already interpolated

  keyTakeaways: string;
  areasOfConcern: string;
  aiPending: string;
  actionItems: string;
  assignedTo: string;     // "Assigned to: {assigneeName}" — caller interpolates
  due: string;            // "Due: {dueDate}" — caller interpolates
  managerInsights: string;
  coachingSuggestions: string;
  riskIndicators: string;
  button: string;
  footer: string;
}

interface SessionSummaryEmailProps {
  // Keep existing dynamic data props:
  variant: "manager" | "report";
  recipientName: string;
  otherPartyName: string;
  sessionNumber: number;
  sessionScore: number | null;
  aiSummary: AiSummary | null;
  actionItems: ActionItem[];
  viewSessionUrl: string;
  aiAddendum?: AiAddendum | null;
  // Add labels prop:
  labels: SessionSummaryLabels;
}
```

In the JSX, replace every hardcoded string with its corresponding `labels.*` prop:
- `<Text style={heading}>Session #{sessionNumber} Summary</Text>` → `<Text style={heading}>{labels.heading}</Text>`
- `Hi {recipientName}, here is the summary...` → `{labels.greeting}`
- `Score: {sessionScore.toFixed(1)} / 5.0` → `{labels.score}`
- `"Key Takeaways"` → `{labels.keyTakeaways}`
- `"Areas of Concern"` → `{labels.areasOfConcern}`
- `"AI summary is being generated..."` → `{labels.aiPending}`
- `"Action Items"` → `{labels.actionItems}`
- `Assigned to: {item.assigneeName}` → `{labels.assignedTo}` (caller pre-interpolates)
- `Due: ${item.dueDate}` → `{item.dueDate ? labels.due : ""}` (caller pre-interpolates the due string)
- `"Manager Insights"` → `{labels.managerInsights}`
- `"Coaching Suggestions"` → `{labels.coachingSuggestions}`
- `"Risk Indicators"` → `{labels.riskIndicators}`
- `"View Full Session"` → `{labels.button}`
- Pass `labels.footer` to `<EmailLayout footerText={labels.footer}>`

For `assignedTo` and `due`: the call site will build the full string using `t("emails.sessionSummary.assignedTo", { assigneeName })` and `t("emails.sessionSummary.due", { dueDate })`. The template receives these as pre-interpolated strings in the `ActionItem` type. Extend `ActionItem`:

```typescript
interface ActionItem {

  dueDate: string | null;

  dueLabel: string | null;   // pre-interpolated "Due: date" or null
}
```
In JSX: `<Text style={metadataRow}>{item.assignedToLabel}{item.dueLabel ? ` | ${item.dueLabel}` : ""}</Text>`
