    1. Create `src/lib/templates/version-diff.ts`:
       - Export type `VersionChange = { type: "added" | "removed" | "modified"; entity: "section" | "question"; name: string; details?: string; }`
       - Export `function computeVersionDiff(oldSnapshot: TemplateVersionSnapshot, newSnapshot: TemplateVersionSnapshot): VersionChange[]`
       - Algorithm:
         a. Build Map of old sections by id, new sections by id
         b. For sections in new but not old: push `{ type: "added", entity: "section", name: s.name }`
         c. For sections in old but not new: push `{ type: "removed", entity: "section", name: s.name }`
         d. For sections in both: compare name (if changed: push modified section). Then compare questions:
            - Build Map of old questions by id, new questions by id (within that section)
            - Questions in new but not old: `{ type: "added", entity: "question", name: q.questionText }`
            - Questions in old but not new: `{ type: "removed", entity: "question", name: q.questionText }`
            - Questions in both: compare `questionText`, `answerType`, `answerConfig` (JSON.stringify), `isRequired`, `helpText`. If any differ: `{ type: "modified", entity: "question", name: q.questionText, details: "text changed" | "answer type changed" | "multiple changes" }`

    2. Parse versionNumber: `const vNum = parseInt(versionNumber, 10)`

    3. Inside `withTenantContext`:
       a. Verify template exists and is not archived
       b. Fetch the version snapshot row (404 if not found)
       c. Cast snapshot: `const snap = version.snapshot as TemplateVersionSnapshot`

       d. Archive ALL current non-archived sections:
          ```typescript
          await tx.update(templateSections)
            .set({ isArchived: true })
            .where(and(
              eq(templateSections.templateId, id),
              eq(templateSections.isArchived, false)
            ));
          ```

       e. Archive ALL current non-archived questions:
          ```typescript
          await tx.update(templateQuestions)
            .set({ isArchived: true })
            .where(and(
              eq(templateQuestions.templateId, id),
              eq(templateQuestions.isArchived, false)
            ));
          ```

       f. Delete current label assignments:
          ```typescript
          await tx.delete(templateLabelAssignments)
            .where(eq(templateLabelAssignments.templateId, id));
          ```

       g. Insert new sections from snapshot, building oldSectionId -> newSectionId map:
          ```typescript
          const sectionMap = new Map<string, string>();
          for (const s of snap.sections) {
            const [newSection] = await tx.insert(templateSections).values({
              templateId: id,
              tenantId: session.user.tenantId,
              name: s.name,
              description: s.description,
              sortOrder: s.sortOrder,
              isArchived: false,
            }).returning();
            sectionMap.set(s.id, newSection.id);
          }
          ```

       h. Insert new questions from snapshot (two-pass for conditional remapping, same as duplicate route):
          - First pass: insert all questions with `conditionalOnQuestionId: null`, building `oldQuestionId -> newQuestionId` map
          - Second pass: update `conditionalOnQuestionId` using the map for questions that had conditional references in the snapshot

       i. Re-insert label assignments from snapshot `labelIds` array:
          ```typescript
          if (snap.labelIds.length > 0) {
            await tx.insert(templateLabelAssignments).values(
              snap.labelIds.map(labelId => ({ templateId: id, labelId }))
            );
          }
          ```

       j. Set template as unpublished draft:
          ```typescript
          await tx.update(questionnaireTemplates)
            .set({ isPublished: false, updatedAt: new Date() })
            .where(eq(questionnaireTemplates.id, id));
          ```

       k. Audit log with action `"template_version_restored"`, metadata: `{ versionNumber: vNum, templateName: template.name }`
