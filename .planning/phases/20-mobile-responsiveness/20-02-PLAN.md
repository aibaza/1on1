**Step A — Add controlled open props to ImportDialog:**

In `import-dialog.tsx`, extend `ImportDialogProps` to accept optional controlled state:

```tsx
interface ImportDialogProps {
  currentUserRole: string;
  contentLanguage: string;
  onImportSuccess?: () => void;
  // New optional controlled props:
  open?: boolean;
  onOpenChange?: (open: boolean) => void;
}
```

In the component body, use controlled state when props are provided:
```tsx
const [internalOpen, setInternalOpen] = useState(false);
const isControlled = open !== undefined;
const dialogOpen = isControlled ? open : internalOpen;
const setDialogOpen = isControlled ? (onOpenChange ?? (() => {})) : setInternalOpen;
```

Replace all `setOpen(...)` calls with `setDialogOpen(...)` and `open` state references with `dialogOpen`.

Keep the existing trigger button inside ImportDialog unchanged — it still controls `internalOpen` when not controlled from outside. This means the desktop layout continues to work with zero changes.
