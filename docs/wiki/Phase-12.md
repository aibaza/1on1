**Status**: Complete
**Milestone**: v1.1 Internationalization
**Depends on**: Phase 11
**Completed**: 2026-03-06

## Goal

Every user-facing page and component displays translated strings, locale-aware formatting, and translated validation errors â€” the entire UI works in any supported language. Plus a series card redesign delivering richer at-a-glance context.

## Success Criteria

1. User can open a language picker in the user menu, select Romanian, and see the entire UI update without a full page reload â€” preference persists across sessions
2. All pages display translated strings with no hardcoded English visible: auth pages, dashboard, session wizard, people/teams, templates, analytics, settings, and command palette
3. A Romanian-locale user sees dates as "05.03.2026", numbers as "1.234,56", and relative times as "acum 3 zile" â€” an English-locale user sees US formats â€” consistently across all pages including analytics chart axes, tooltips, and labels
4. A Romanian-locale user submitting an invalid form sees validation errors in Romanian; API error responses display in the user's UI language in toast notifications

## Post-Phase Improvements (same session)

After Phase 12 verification, the series card was further improved with visual richness:

- **Star rating display** â€” amber filled stars for scored sessions (half-star support), gray empty stars when no rating exists; numeric score appears on hover
- **Score history sparkline background** â€” subtle area chart using `--chart-1` color fills the bottom 30% of each series card, giving a historical trend at a glance
- **AI session summary on cards** â€” replaced pre-session nudge preview with post-session `cardBlurb` (1-2 sentence plain-language summary) extracted from the latest completed session's `aiSummary` JSON; colored sentiment dot: green (positive), amber (neutral/mixed), red (concerning); grayed italic placeholder before first session; `cardBlurb` field added to `AISummary` schema and generated by the AI pipeline at session completion
- **Nudge modal** â€” session card click on dashboard opens a coaching nudges modal sorted by priority with dismiss support (nudges no longer shown inline on cards)

## Key Decisions

- Zod v4 error map adapted: single-arg signature, `'origin'` not `'type'`, `'invalid_format'` not `'invalid_string'`
- Zod `.refine()` messages remain English (schemas are language-agnostic)
- `useFormatter()` called at component level, captured in closures for Recharts callbacks
- `format.number()` replaces all `.toFixed()` for locale-aware decimal display
- Dynamic translation key lookup uses `as any` cast for TypeScript strict mode compatibility
- Mood emoji placeholders kept as English defaults (user-customizable content, not UI chrome)
- `cardBlurb` in `AISummary` schema â€” old sessions without this field show the placeholder gracefully
- Series card sentiment mapping: `overallSentiment` values "neutral" and "mixed" both map to amber dot
